## Tool Selection

| Factor | GitHub Copilot | Cursor | Claude Code |
|--------|---------------|--------|-------------|
| **Enterprise adoption** | Excellent | Growing | Limited |
| **IDE support** | VS Code, JetBrains | VS Code fork | Terminal |
| **Context window** | Standard | Large | Very large |
| **Agent capabilities** | Basic | Advanced | Advanced |
| **Best for** | Code completion | Codebase exploration | Complex reasoning |

## Common Configuration

All tools benefit from:
- `.cursor/rules/` or equivalent for project-specific guidance
- Agent instruction files for consistent behavior
- Building block definitions for pattern selection
