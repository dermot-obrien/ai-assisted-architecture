## How to Contribute

### Priority Contributions Needed

1. **Methodology feedback** - Practical experience applying the methodology
2. **Building blocks** - Reference ABBs and SBBs
3. **Tool guides** - Deep dives on specific AI tools
4. **Industry patterns** - Sector-specific adaptations

### Contribution Process

1. Open an Issue describing the contribution
2. Discuss with maintainer
3. Fork and develop
4. Submit PR with clear description

See [CONTRIBUTING.md](https://github.com/dermot-obrien/ai-assisted-architecture/blob/main/CONTRIBUTING.md) for details.
