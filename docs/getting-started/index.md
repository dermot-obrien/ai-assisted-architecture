## Prerequisites

Before you begin, ensure you have:

| Requirement | Description |
|-------------|-------------|
| **IDE** | VS Code, Cursor, or IntelliJ IDEA |
| **AI Tool** | GitHub Copilot, Cursor AI, or Claude Code |
| **Git** | Version control for architecture repositories |
| **Python** (optional) | For document conversion scripts |
