# AI-Assisted Architecture

Welcome to AI-Assisted Architecture — an opinionated methodology for leveraging AI agents in enterprise and solution architecture practice.

## What is AI-Assisted Architecture?

AI-Assisted Architecture is a framework that enables architects to:

- **Accelerate research** through AI-powered synthesis of technology landscapes
- **Draft documentation** with AI assistance while maintaining human accountability
- **Apply patterns** consistently using AI agent recommendations
- **Track provenance** of AI-generated content for governance and audit

Organizations can adopt this framework through a fork-and-customize model:

1. **Fork** the core repository
2. **Customize** templates, branding, and governance
3. **Extend** with organization-specific building blocks
4. **Sync** updates from upstream
5. **Contribute** improvements back to the community
